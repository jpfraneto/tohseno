APACHE-2.0 / YOUR MACHINE / YOUR REPOSITORIES

Open underneath.
Magic at the surface.

Every shot begins from a working SwiftUI base and lands as its own Git repository—with source, history, tests, manifest, local data, and a clean way out.

The one-line installer authenticates the release and its complete file inventory. The launcher checks the installed CLI and managed runtime again before every run.

Studio opens through an owner-only local launcher. Every coding-agent exit is followed by privacy and integrity verification; an unsafe result is isolated instead of called ready.

No TOHSENO account. No TOHSENO credentials. No backend receiving your users’ content. Your selected coding agent remains a separate provider trust boundary.
